# LLM Hallucination Detector

Репозиторий содержит решение задачи по выявлению фактических галлюцинаций LLM модели, созданное во время участия в хакатоне IT Purple Hack 2026.

## Структура репозитория

### datasets
* **публичный датасет** — исходные данные для валидации.
* **main_train_dataset.csv** — итоговый тренировочный датасет, на котором обучалась модель.

### data_pipeline
В этой папке содержатся инструменты для подготовки данных:
* **Парсеры и скрипты:** инструменты для сбора статей, ставших фундаментом для генерации вопросов.
* **Генерация вопросов:** ноутбук с логикой создания пар «вопрос-ответ» на базе собранных статей с использованием модели **GigaChat**.
* **Сбор ответов:** ноутбук для получения ответов целевой модели на сгенерированные вопросы.

### model_and_features
* **Основной ноутбук:** содержит полный цикл Feature Engineering и обучения классификатора.
* **Feature Files:** готовые файлы с извлеченными признаками.

##  Сбор данных и создание датасета

При разработке модели было уделено достаточно внимания качеству обучающей выборки:
1. **Анализ данных:** Проанализирован публичный датасет и выделен основной пул тем.
2. **Сбор источников:** Созданы парсеры для поиска статей в различных источниках, включая Википедию.
3. **Разнообразие вопросов:** Было разработано несколько типов промптов для генерации вопросов различного вида.
4. **Вероятностное распределение:** Для каждой категории тем настроено специфическое распределение типов промптов, чтобы максимизировать шанс выявления галлюцинаций.
5. **Контекст:** В датасете `main_train_dataset.csv` сохранен столбец `context` с конкретными фрагментами текстов, на которых строились вопросы.

## Извлечение признаков, Feature Engineering
Значительно расширен стандартный набор признаков, добавлен глубокий анализ внутренних состояний модели:
* **Attention Features:** через хуки на слои внимания были собраны энтропия, сумма весов внимания, максимальные веса, средние значения и стандартные отклонения.
* **Logit Analysis:**  Разница вероятностей между top-1 и top-2 токенами.
    * Семантическая близость топ-5 предсказаний.
    * Определение слоя, на котором модель окончательно сходится к ответу.
* **Hidden States:** расчет косинусного и евклидова расстояния между скрытыми состояниями слоев.
* **Динамика энтропии:** отслеживание изменения энтропии от первого к последнему слою и определение слоя с минимальной энтропией.

## Обучение модели

Финальный процесс обучения был построен на базе библиотеки **CatBoost**, с помощью нее был проведен отбор наиболее значимых признаков и произведено финальное обучение на оптимизированном наборе фичей.

Итоговый PR-AUC составил 0.828. Что принесло 8/70 место в соревновании.
